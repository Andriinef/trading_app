# FastApi

## Install FastApi

``` code
pipenv install fastapi[all]
```

## Start the FastApi

``` code
uvicorn main:app --reload
```

## Request Files

``` code
pipenv install python-multipart
```

## Sqlalchemy

```code
pipenv install sqlalchemy
```

## Alembic

```code
pipenv install alembic
```

## Postgresql

```code
pipenv install psycopg2-binary
```

``` code
pipenv install ormar[postgresql]
```
